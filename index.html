RespawnIRC

Un JVIrc-like

Présentation

RespawnIRC est un logiciel offrant une interface de client IRC pour les topics de jeuxvideo.com.
Pour plus d'informations, veuillez consulter le message de présentation sur JVC : http://www.jeuxvideo.com/respawnirc/forums/message/731107420

Prochainement...

Le prochain ajout sera la récupération du 1er message d'un topic.

FAQ

Pourquoi faut-il sélectionner le texte se trouvant dans les balises spoils et pas juste passé sa souris dessus comme sur les forums de steam par exemple ?

Actuellement avec le widget (composant d'interface graphique) qu'utilise RespawnIRC il est impossible de faire une telle chose, en remplaçant par un widget plus "puissant" ça serait possible, mais ça consommerait aussi plus, que ce soit en terme de mémoire ou de processeur.

Est-ce que les captcha reviendront pour poster/éditer un message ou ça sera désactivé à tout jamais ?

Aucune idée pour l'instant, mais ce n'est pas prévu qu'ils reviennent.

Est-ce qu'un outil pour mettre à jour automatiquement RespawnIRC est prévu ?

Non, pas pour l'instant.

Les stickers sont trop gros/sont moche après que leur taille ai été réduite, que faire ?

Pour réduire la taille de stickers via le logiciel il faut aller dans le menu configuration > taille des stickers, mais ils peuvent être assez moches après ça, donc je conseil plutôt de changer en plus de ça la taille des images d'origine qui se trouvent dans le dossier "ressources/stickers/"

Les smileys ne sont pas animé, est-ce normal ?

Oui c'est normal, leur animation est prévue mais n'est pas prioritaire.

Comment créer un thème, y a-t-il un outil pour simplifier leurs créations ?

Pour créer un thème il suffit de lire la page consacrée à la création de thème sur le wiki (https://github.com/LEpigeon888/RespawnIRC/wiki/Créer-un-thème). Par contre aucun outil n'existe pour simplifier leurs créations.
